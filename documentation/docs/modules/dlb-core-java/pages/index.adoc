== Introduction
The "Core" of the Dialogue Branch software is a library that can read and execute Dialogue Branch (.dlb) scripts. This core library is written in Java, and can be found on the GitHub repository under https://github.com/dialoguebranch/dlb-core-java[dlb-core-java].
