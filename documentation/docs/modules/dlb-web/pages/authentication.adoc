== Native vs Keycloak
The Dialogue Branch Web Service supports two ways in which Authorization is handled: Native or Keycloak.

The Native Authentication Service is a simple built-in user management system that allows you to quickly get started with using the Dialogue Branch Web Service, but is not recommended for production environments.

The Keycloak Authentication Service uses `https://www.keycloak.org/[Keycloak]`, an Open Source Identity and Access Management platform, to handle user management and the generation of access tokens.
